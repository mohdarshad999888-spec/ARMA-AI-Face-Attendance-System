# ARMA AI Face Attendance System

An AI-powered Face Attendance System built using Flask, Firebase, OpenCV, HTML/CSS, JavaScript, and Machine Learning.

---

# 🚀 Features

- AI Face Recognition
- Real-time Attendance Marking
- Firebase Realtime Database Integration
- Duplicate Attendance Prevention
- Duplicate Registration Protection
- Camera-Based Face Registration
- Modern Futuristic Dashboard UI
- Smart Face Alignment Detection
- Automatic Model Training
- Live Status Feedback

---

# 🛠 Tech Stack

- Python
- Flask
- OpenCV
- Firebase
- HTML
- CSS
- JavaScript
- Machine Learning
- KNN Classifier

---

# 📸 Modules

## 🔹 Student Registration
Register students using live camera capture with guided face alignment.

## 🔹 Face Attendance
Automatically detect and mark attendance using AI face recognition.

## 🔹 Reports
View attendance records stored in Firebase.
